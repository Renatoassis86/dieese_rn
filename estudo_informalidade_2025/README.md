# Estudo da Informalidade no RN — Visualização com Design Thinking e Storytelling

Este projeto foi gerado automaticamente.
Estrutura alinhada às aulas de Design Thinking, Storytelling com Dados e Visualização (Matplotlib).

## Como rodar
1. Ative seu ambiente e instale dependências:
   pip install -r requirements.txt
2. Execute os scripts em `scripts/`.
3. As figuras serão salvas em `outputs/figs/`.
