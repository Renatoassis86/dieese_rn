# Dicionário de Dados - Censo Agropecuário 2017 (IBGE)

Este dicionário lista as principais tabelas do SIDRA que serão utilizadas para o diagnóstico da Agricultura Familiar no RN.

## 1. Estrutura dos Estabelecimentos (Chave: Agricultura Familiar vs. Não-Familiar)

| Tabela SIDRA | Descrição | Principais Variáveis |
| :--- | :--- | :--- |
| **6778** | Número de estabelecimentos e Área | Total de est., Área total (hectares). |
| **6779** | Condição do produtor em relação às terras | Proprietário, Arrendatário, Parceiro, Ocupante. |
| **6784** | Grupos de atividade econômica | Lavoura (temp/perm), Pecuária, Horticultura, Florestal. |

## 2. Pessoal e Mão de Obra

| Tabela SIDRA | Descrição | Principais Variáveis |
| :--- | :--- | :--- |
| **6190** | Pessoal ocupado nos estabelecimentos | Com vínculo de parentesco vs. Sem vínculo. |
| **6792** | Pessoal ocupado por sexo e idade | Homens/Mulheres, Faixas etárias (jovens/idosos). |

## 3. Produção e Tecnologia

| Tabela SIDRA | Descrição | Principais Variáveis |
| :--- | :--- | :--- |
| **6884** | Valor da produção | Valor total, Valor por grupo de atividade. |
| **6876** | Uso de irrigação | Área irrigada, Métodos de irrigação. |
| **6871** | Uso de agrotóxicos | Estabelecimentos que utilizaram vs. Não utilizaram. |
| **6880** | Acesso a crédito | Origem do crédito (Pronaf, Bancos, etc.). |
| **6882** | Orientação técnica | Recebeu assistência técnica (pública, privada, própria). |

## 4. Integração Territorial
*   **Geografia**: Os dados serão extraídos para o nível de **Municípios (N6)** do Rio Grande do Norte (UF 24).
*   **Abertura**: Sempre que disponível, os dados serão filtrados pela variável **"Tipologia (Agricultura Familiar / Não Familiar)"**.
